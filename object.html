Global Objects

Global objects in JavaScript are objects that are available globally in all parts of your JavaScript code, whether it's running in a web browser, a Node.js environment, or another JavaScript runtime environment. These objects provide built-in functionalities that you can use without needing to define or import them explicitly.

Some of the most used global Objects include :- Object, Function, String, Math, Date, console, JSON, RegEx,Number e.t.c.
